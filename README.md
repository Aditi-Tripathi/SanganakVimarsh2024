**Sanganak Vimarsh 2024 – Event Management Framework**

*Problem Statement*

As the Digital Head of Tech-A-Thon, the objective was to design and develop a structured framework to efficiently manage:

1. Event registrations
2. Participant queries
3. Sponsor showcase
4. Event gallery
5. Information dissemination
 
for the annual technical fest Sanganak Vimarsh 2024.

*Tech Stack*

1. HTML5
2. CSS
3. JavaScript

*Features*

1️. Event Registration Module

User-friendly registration form
Input validation
Structured data handling
Confirmation feedback to users

2. Query Management Section

Dedicated contact/query form
Organized information structure for easy resolution
Clear communication flow

3️. Sponsor Showcase

Dynamic sponsor display section
Categorized sponsor representation
Visual branding integration

4️. Gallery Section

Event highlights display
Structured image rendering
Responsive layout

5️. Information Portal

Event schedule
Team details
Contact information
About section

*System Design Approach*

Modular file structure
Separation of concerns (HTML structure, CSS styling, JS logic)
Responsive design principles
Clean UI for accessibility
